# Hello World Python Program

A simple Python program that prints "hello_world" to the console.

## Requirements

- Python 3.11 or higher

## Installation

No external dependencies are required. Simply ensure you have Python installed:

```bash
python --version
```

## Usage

Run the program directly:

```bash
python hello_world.py
```

Or make it executable and run:

```bash
chmod +x hello_world.py
./hello_world.py
```

## Expected Output

```
hello_world
```

## Testing

Run the unit tests:

```bash
python -m pytest test_hello_world.py
```

Or using unittest:

```bash
python -m unittest test_hello_world.py
```
