# Hello World Python Program

A simple Python program that prints "Hello World" to the console.

## Requirements

- Python 3.11 or higher

## Installation

No additional dependencies are required. This program uses only Python's standard library.

## Usage

Run the program using Python:

```bash
python hello.py
```

Or make it executable and run directly (Unix/Linux/macOS):

```bash
chmod +x hello.py
./hello.py
```

## Expected Output

```
Hello World
```
