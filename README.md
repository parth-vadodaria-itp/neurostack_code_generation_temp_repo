# Hello 2046 Python Program

A simple Python program that prints "Hello 2046" to the console.

## Requirements

- Python 3.11 or higher

## Installation

No external dependencies are required. Python 3.11+ is sufficient.

```bash
# Optional: Create a virtual environment
python3 -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate

# Optional: Install dependencies (none required for this program)
pip install -r requirements.txt
```

## Usage

Run the program using Python:

```bash
python hello.py
```

Or make it executable and run directly:

```bash
chmod +x hello.py
./hello.py
```

## Expected Output

```
Hello 2046
```

## Testing

You can verify the output using:

```bash
python hello.py
```

The program should print exactly: `Hello 2046`
