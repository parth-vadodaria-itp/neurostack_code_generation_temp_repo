# Contributing to Hello World Program

## Development Workflow

### Prerequisites
- Java 17 or higher installed
- Maven 3.6+ (optional, for build automation)
- Git

### Setup
1. Clone the repository:
   ```bash
   git clone https://github.com/parth-vadodaria-itp/neurostack_code_generation_temp_repo.git
   cd neurostack_code_generation_temp_repo
   ```

2. Compile and run:
   ```bash
   javac HelloWorld.java
   java HelloWorld
   ```

### Making Changes
1. Create a feature branch:
   ```bash
   git checkout -b feature/your-feature-name
   ```

2. Make your changes to `HelloWorld.java`

3. Test your changes:
   ```bash
   javac HelloWorld.java && java HelloWorld
   ```

4. Commit and push:
   ```bash
   git add .
   git commit -m "Description of changes"
   git push origin feature/your-feature-name
   ```

5. Create a Pull Request on GitHub

## Code Standards
- Follow Java naming conventions
- Keep the program simple and focused
- Ensure output remains "Hello World"
- Add comments for any complex logic (though this program is intentionally simple)

## Testing
Run the program and verify output:
```bash
java HelloWorld
# Expected output: Hello World
```
